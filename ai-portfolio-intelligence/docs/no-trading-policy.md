# No-Trading Policy

The system is a read-only portfolio analyst.

Forbidden capabilities:

- Placing, modifying, cancelling, routing, or submitting orders
- Executing trades
- Auto-rebalancing
- Scheduled broker activity
- One-click broker execution controls
- Automated stop-loss, take-profit, equity, ETF, or option order activity

Allowed language:

- Consider
- Review
- Watch
- Potential add zone
- Potential trim zone
- Exit review trigger
- Risk alert
- Decision-support suggestion
- Human review required

Every generated report must include:

> This is portfolio analysis and decision support only. The system does not execute trades. The user must independently review any suggestion before making investment decisions outside the platform.
