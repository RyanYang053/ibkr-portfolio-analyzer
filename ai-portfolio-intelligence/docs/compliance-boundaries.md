# Compliance Boundaries

The system provides portfolio research, risk monitoring, structured reports, and decision-support suggestions.

It is not a trading bot, robo-advisor, execution platform, order management system, or guaranteed-profit tool.

User-facing output must:

- Separate facts from interpretation
- Disclose mock, missing, stale, or unavailable data
- Avoid guarantees
- Avoid `must buy` and `must sell` language
- Require independent human review
- State that the platform does not execute trades
