# API

Implemented endpoint groups:

- `/auth`
- `/broker` read-only
- `/portfolio`
- `/stocks`
- `/recommendations`
- `/reports`
- `/watchlist`
- `/alerts`
- `/admin/audit-logs`

Forbidden endpoint groups are intentionally absent:

- `/orders/*`
- `/trades/*`
- `/rebalance/execute`
- `/options/execute`
- `/broker/order`
- `/broker/trade`
