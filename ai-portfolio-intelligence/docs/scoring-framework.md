# Scoring Framework

Scores are normalized from 0 to 100 and interpreted as:

- 85-100: Excellent
- 70-84: Good
- 55-69: Mixed
- 40-54: Weak
- Below 40: Broken/high-risk

Implemented stock-type models:

- Universal
- Mega-cap quality
- ETF
- Speculative growth

Each recommendation includes a score, confidence level, evidence list, missing data assumptions, add zone, hold zone, trim review zone, exit review trigger, and human review reminder.
